Optional绑定

使用`Optional`绑定来确定`Optional`项是否包含值，如果是，则使该值可用作临时常量或变量。

var myString:String? myString = "Hello, Swift 4!" if let yourString = myString {   print("Your string has - \(yourString)") } else {   print("Your string does not have a value") }
