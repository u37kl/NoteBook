# 常用的字符实体

> 字符实体其实就是转义字符

| 结果 | 描述                        | 实体名称   | 实体编号 |
| :--- | :-------------------------- | :--------- | :------- |
| "    | quotation mark              | `&quot;`   | `&#34;`  |
| '    | apostrophe                  | `&apos;`   | `&#39;`  |
| &    | ampersand                   | `&amp;`    | `&#38;`  |
| <    | less-than                   | `&lt;`     | `&#60;`  |
| >    | greater-than                | `&gt;`     | `&#62;`  |
| ¥    | yen                         | `&yen;`    | `&#165;` |
| §    | section                     | `&sect;`   | `&#167;` |
| ©    | copyright                   | `&copy;`   | `&#169;` |
| ®    | registered trademark        | `&reg;`    | `&#174;` |
| ±    | plus-or-minus               | `&plusmn;` | `&#177;` |
| ×    | multiplication              | `&times;`  | `&#215;` |
| ÷    | division                    | `&divide;` | `&#247;` |
| ·    | middle dot                  | `&middot;` | `&#183;` |
| º    | masculine ordinal indicator | `&ordm;`   | `&#186;` |
